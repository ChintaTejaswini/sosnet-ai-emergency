# 🚨 SOSNet – AI Emergency Rescue Platform

SOSNet is an **AI-powered emergency rescue platform** designed to help users trigger SOS alerts instantly using **voice, tap, location, and health monitoring features**.

Built for **real-time emergency situations**, SOSNet focuses on **speed, accessibility, and reliability**, especially when users are unable to type or manually explain their situation.

---

❗ Problem Statement

In real emergency situations, victims often cannot unlock their phone, type messages, explain their location, or manually call for help due to panic, injury, weak network, or time pressure.

Traditional SOS apps depend too much on manual interaction, which can fail in critical moments.

SOSNet solves this by enabling hands-free voice-triggered emergency alerts with instant location and health context, making rescue response faster and smarter.

----
 
# ✨ Features

* 🎤 **Voice Activated SOS**
  Users can simply say emergency keywords like **“help”, “accident”, “SOS”, “danger”** and the system automatically detects and triggers alerts.

* 📍 **Live Location Tracking**
  Shares real-time user location with emergency contacts and responders.

* ❤️ **Heartbeat / Health Monitoring**
  Uses fingerprint sensor camera logic to estimate heartbeat and detect abnormal conditions.

* 📞 **One Tap Emergency Alert**
  Quick emergency button for instant SOS activation.

* 📜 **User History**
  Stores previous emergency events and alert logs.

* 🌐 **Weak Network Friendly**
  Optimized to work in low-signal situations.

* 📱 **Responsive Web App UI**
  Works smoothly on mobile and desktop browsers.

# 💡 How It Works

1. User opens SOSNet
2. Voice listener starts automatically
3. User says an emergency phrase
4. Speech recognition converts it into text
5. SOS logic validates emergency intent
6. Live location + health data captured
7. Emergency alert is sent instantly
